# probot-jser-info

> a GitHub App built with [probot](https://github.com/probot/probot) for [JSer.info](https://jser.info/). 

- <https://github.com/jser/jser.github.io>
- <https://github.com/apps/probot-for-jser-info>

## Setup

```sh
# Install dependencies
npm install

# Run the bot
npm start
```

## Deploy

    ./deploy.sh
